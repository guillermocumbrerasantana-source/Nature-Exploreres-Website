# Nature Explorers - Web Project

## Description

This project is a responsive website about nature exploration and outdoor adventures. The web presents information about expeditions, equipment, and features a beautiful gallery of natural landscapes using local images.

## Features

- Fully responsive design with media queries for 4 device sizes
- Adaptive header and footer
- Image gallery with 10+ local nature images and hover effects
- CRUD system for managing expeditions
- Contact form with JavaScript validation
- Multiple informative sections about nature adventures
- Animation effects and transitions
- Smooth navigation between sections
- Hamburger menu for mobile devices
- Local image management

## Technologies Used

- HTML5
- CSS3 (with CSS variables, Flexbox and Grid)
- Vanilla JavaScript
- Font Awesome for icons
- LocalStorage for data persistence

## Installation and Usage

1. Clone or download the project files
2. Ensure all image files are in the `img/` folder
3. Open the `index.html` file in your browser
4. Explore the website and manage expeditions in the dedicated section
